# Day 3

## What I Did

* Performed EDA on Titanic dataset
* Built a basic Logistic Regression model

## Dataset

https://www.kaggle.com/c/titanic/data

## Key Findings

* Females had higher survival rate
* Young passengers had higher survival rate than older passangers
* Higher passenger class → higher survival
* Age had missing values → filled with median

## Model

* Logistic Regression
* Used features: Pclass, Sex, Age

## Accuracy

~0.77 (approx)

## Key Learning

* Importance of feature selection
* Handling missing values
* Basic model pipeline
